# EMNIST Letter Detective

This project demonstrates handwritten letter recognition using the **EMNIST Letters** dataset.  
The notebook **EMNIST Letter Detective.ipynb** includes data preprocessing, CNN model training, evaluation, and predictions.

## Overview
- Uses EMNIST Letters (26 classes A–Z)
- 28×28 grayscale handwritten characters
- CNN-based classification model
- Training, visualization, and testing all in one notebook
